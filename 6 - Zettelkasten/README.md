# 6 - Zettelkasten

**The garden.** Permanent notes — atomic, self-contained, linked.

> The manifesto for *why* this exists lives in [[6 - Zettelkasten/About|About]].
> The dashboard for *managing* it lives in [[0 - Meta/Garden]].

## What Belongs Here

Only **permanent notes**. A permanent note:

1. **Atomic** — expresses *one* idea.
2. **Self-contained** — understandable without the source it came from.
3. **In your own words** — not a quote, not a summary; a thought you've internalized and re-expressed.
4. **Connected** — links to at least one other note, with an explanation of the relationship.

If a note isn't yet all four, it's a [[1 - Rough Notes/README|fleeting note]], a reading note inside a [[2 - Source Materials/README|source]], or a seedling on its way to becoming permanent.

## Conventions

- **Template:** [[5 - Templates/Main Notes]] (auto-applied when creating a file here).
- **Filename:** descriptive title — what the idea *is*, in your own words. Not a tag, not a question, not the source.
  - ✓ `Fortune favors the prepared mind.md`
  - ✗ `Cardano on luck.md` (source-coupled — what happens if you find the idea elsewhere?)
- **Maturity status** (frontmatter): `seedling` → `budding` → `evergreen` → `archived`. See the [[0 - Meta/Workflows#Maturity Ladder|maturity ladder]].
- **Connections matter more than links.** A wiki-link without an explanation is half a note.
