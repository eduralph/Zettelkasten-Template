# Vault Overview

A personal Zettelkasten and writing project — a "book of insights" for my sons (and anyone else who finds it useful). See [[About]] for the manifesto.

# Folder Structure

| Folder | Purpose |
|---|---|
| **[[0 - Meta/README\|0 - Meta]]** | Documentation about the vault itself — conventions, workflows, dashboards. The control panel. |
| **[[1 - Rough Notes/README\|1 - Rough Notes]]** | Inbox / fleeting notes. Raw capture, not yet processed. |
| **[[2 - Source Materials/README\|2 - Source Materials]]** | Books, articles, podcasts, lectures — anything I'm consuming. Subfolders by medium (e.g., `Books/`). |
| **[[3 - Tags/README\|3 - Tags]]** | Tag descriptions / glossary. What each tag means and when to use it. |
| **[[4 - Indexes/README\|4 - Indexes]]** | Hand-curated maps of content — entry points into the vault by topic. |
| **[[5 - Templates/README\|5 - Templates]]** | Templater templates used to scaffold new notes. |
| **[[6 - Zettelkasten/README\|6 - Zettelkasten]]** | The garden. Permanent notes — atomic, self-contained, linked. |

# Meta Dashboard

- [[0 - Meta/Garden|🌱 Garden]] — Zettelkasten dashboard (seedlings, orphans, stale notes, census)
- [[0 - Meta/Reading List|📚 Reading List]] — All books, grouped by status
- [[0 - Meta/Folder Conventions|📂 Folder Conventions]] — Naming rules and folder usage
- [[0 - Meta/Workflows|⚙️ Workflows]] — Capture, process, review rituals
- [[0 - Meta/Plugin Setup|🔌 Plugin Setup]] — Installed plugins and why
- [[0 - Meta/Roadmap|🗺️ Roadmap]] — What's coming next
- [[0 - Meta/Changelog|📝 Changelog]] — System changes over time

# Quick Start

- **New book note** → create file inside `2 - Source Materials/Books/` (Templater auto-populates from `Book Template`)
- **New Zettelkasten note** → create file inside `6 - Zettelkasten/` (Templater auto-populates from `Main Notes`)
- **Review the garden** → open [[0 - Meta/Garden]]
