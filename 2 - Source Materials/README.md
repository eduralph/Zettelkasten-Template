# 2 - Source Materials

**External inputs.** Books, articles, podcasts, lectures, talks — anything I'm consuming and want to take notes on.

## Purpose

- One note per source.
- Reading notes (quotes, reactions, chapter notes) live inside the source note.
- When a passage triggers an *original idea*, that idea graduates to a permanent note in [[6 - Zettelkasten/About|6 - Zettelkasten]] — linking back to the source here.

## Subfolders

| Subfolder | Medium | Template |
|---|---|---|
| `Books/` | Books | [[5 - Templates/Book Template]] (auto-applied via Templater) |

Add new media subfolders as they become relevant (`Articles/`, `Podcasts/`, `Lectures/`, etc.) and create a matching Templater folder mapping in [[0 - Meta/Plugin Setup]].

## Conventions

- Filename = title of the work (e.g., `De vita propria liber.md`, not `cardano-autobiography.md`).
- Frontmatter `status` values: `to-read`, `reading`, `finished`. Keep these consistent so [[0 - Meta/Reading List]] queries work.
